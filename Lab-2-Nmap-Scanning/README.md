# Lab 2 — Nmap Scanning Basics

## Objective

The objective of this lab is to learn basic network scanning and enumeration using Nmap. This lab covers host discovery, port scanning, service detection, and operating system detection.

---

## Tools Used

- Kali Linux
- Nmap

---

## Target

scanme.nmap.org

---

## Tasks Performed

- Host discovery
- TCP port scanning
- Service version detection
- Operating system detection
- Aggressive scanning
- Port enumeration

---

## Skills Learned

- Basic network reconnaissance
- Service identification
- Port scanning techniques
- Enumeration fundamentals
- Nmap command usage

---

## Conclusion

This lab provided hands-on experience with Nmap scanning techniques used in cybersecurity and penetration testing. The lab improved understanding of network services, open ports, and enumeration methods.
